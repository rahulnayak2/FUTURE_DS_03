# 📊 Bank Marketing Campaign — Funnel & Conversion Performance Analysis

## 🔍 About the Project
This project analyzes a bank's telemarketing campaign data to identify 
conversion drop-offs, channel performance, and opportunities to improve 
lead-to-customer conversion.
Completed as part of FutureInterns Data Science & Analytics Internship - Task 3.

## 🛠️ Tools & Technologies Used
- Microsoft Excel — Data Cleaning & PivotTable Analysis
- Power BI — Interactive Dashboard
- DAX — Conversion Rate Measure

## 📁 Dataset
- Source: Bank Marketing Dataset (UCI Machine Learning Repository)
- Records: 41,188 rows and 21 columns

## 🔎 Analysis Performed
- Overall Conversion Rate Analysis
- Marketing Funnel Stage Analysis
- Conversion by Contact Method
- Conversion by Month
- Conversion by Job Type
- Conversion by Education Level

## 💡 Key Insights
- Overall conversion rate of 11.27% from 41,188 contacts
- Cellular channel converts 3x better than telephone (14.7% vs 5.2%)
- Students convert best at 31.4% followed by retired at 25.2%
- December has highest conversion rate at 48.9%
- University educated customers convert at 13.7%
